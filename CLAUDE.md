| 2 | **MCP Servers** | Claude directly queries Supabase, creates GitHub PRs, reads Slack, automates browsers. Turns Claude from "helper" into "teammate." | **Massive** — eliminates copy-paste |
| 3 | **Right skill for your role** | Designer → UI/UX. Writer → marketing. Dev → superpowers. Load the 3–5 that matter. | **High** — better output quality |
| 4 | **Plan mode + subagents** | Plan before building. Subagents for parallel research. Prevents going in circles. | **Medium** — fewer failed attempts |
| 5 | **Good prompting** | "Build X with Y using Z" beats "build X." Context in = quality out. | **Medium** — better first-try results |
