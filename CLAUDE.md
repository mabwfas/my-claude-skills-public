## Core Behaviour Rules

1. **Read this file first** — every session, before any task
2. **Load the right skill** — never skip, always load before starting
3. **Check antigravity** — when no other skill matches
4. **Combine skills** — complex tasks need 2–3 skills
5. **Save important context** — `claude-supermemory:super-save` for cross-session memory
6. **Plan first** — `superpowers:writing-plans` for any task with 3+ steps
7. **Verify before done** — `superpowers:verification-before-completion`
8. **Fix bugs autonomously** — read logs, find root cause, don't ask for hand-holding
9. **Minimal changes** — only touch what's necessary, no unnecessary refactors
10. **Use subagents** — offload research and parallel work to keep context clean
